# SERP Voyager

SERP Voyager is a Streamlit app that utilizes the Google SERP API to obtain search results for specified keywords. The app allows users to customize various parameters, such as location, domain, language, device, and more. Users can input multiple keywords and extract search results for each one, with unique results stored for each keyword.

## Getting Started

These instructions will help you set up and run the SERP Voyager app on your local machine.
